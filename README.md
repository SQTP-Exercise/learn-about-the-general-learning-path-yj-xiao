Help you have a general idea of the learning course. Maybe I'll update unregularly.<br>

## 涉及技术栈
- Node.js & Express.js
- Web & HTTP基础
- RESTful API
- 数据库
- 。。。

## 学习阶段
- 阶段 1：学习后端开发基础（1-2 周）
- 阶段 2：理解 Web 和 HTTP 基础（1 周）
- 阶段 3：学习后端框架和开发数据库（2-3 周）
- 阶段 4：实现用户认证与授权（2 周）
- 阶段 5：实现聊天功能（3-4 周）
- 阶段 6：构建完整项目（2-3 周）（整合所学技术）
- 阶段 7：部署与优化（2 周）


## Weekly project 安排
1. 搭建基本的 Node.js 环境与 Express.js
2. 基础的 RESTful API 开发
3. 学习数据库连接与 CRUD 操作
4. 实现用户注册功能
5. 实现用户登录功能
6. 实现聊天室 API（简易聊天功能）
7. 存储聊天记录到数据库
8. 用户消息通知与前端展示
9. 优化用户体验与性能
10. 部署与上线
